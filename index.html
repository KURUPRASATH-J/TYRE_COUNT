Project Overview

This project implements an intelligent object detection system using YOLOv8 and OpenCV. It processes video streams in real-time, detecting and tracking objects with high accuracy while maintaining the original video FPS.

Real-time Detection

Process videos at original FPS

Easy to Use

Simple Python interface

Prerequisites

System Requirements

Operating System

Windows 10/11, macOS 10.15+, or Linux (Ubuntu 18.04+)

Python Version

Python 3.9 or above (3.10 recommended)

RAM

Minimum 8GB (16GB recommended)

GPU (Optional)

NVIDIA GPU with CUDA support for faster processing

Check Python Installation

Open your terminal/command prompt and run:

python --version

If Python is not installed, download it from python.org

Installation Guide

1

Clone the Repository

First, clone the project from GitHub to your local machine:

git clone https://github.com/KURUPRASATH-J/TYRE_COUNT.git
cd TYRE_COUNT

Don't have Git? Download it here or download the ZIP file from GitHub.

2

Create Virtual Environment (Recommended)

Create an isolated Python environment to avoid dependency conflicts:

# Create virtual environment
python -m venv venv

# Activate virtual environment
venv\Scripts\activate
# Create virtual environment
python3 -m venv venv

# Activate virtual environment
source venv/bin/activate

Your prompt should now show (venv) at the beginning

3

Install Required Packages

Install all necessary Python libraries:

Option 1: Using requirements.txt (Recommended)

pip install -r requirements.txt

Option 2: Manual Installation

# Core packages
pip install ultralytics
pip install opencv-python
pip install numpy
pip install matplotlib
pip install tqdm
pip install pillow

Optional: GPU Support (For NVIDIA GPUs)

pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118

For GPU support, ensure you have CUDA Toolkit installed

4

Verify Installation

Check if all packages are installed correctly:

python -c "import cv2, ultralytics; print('Installation successful!')"

You should see "Installation successful!" if everything is set up correctly

How to Run

Running Object Detection

1. Basic Detection (Default Video)

Run detection on the provided demo video:

python detect.py

2. Detection on Custom Video

Run detection on your own video file:

python detect.py --source path/to/your/video.mp4

3. Live Camera Detection

Run detection using your webcam:

python detect.py --source 0

Use --source 1 for external cameras

4. Advanced Options

# Specify confidence threshold
python detect.py --conf 0.5

# Specify model weights
python detect.py --weights models/best.pt

# Save output video
python detect.py --save --output results/

Troubleshooting

ImportError: No module named 'cv2'

Solution: OpenCV is not installed. Run:

pip install opencv-python
Model file not found error

Solution: Ensure the model file exists in the correct path:

# Check if file exists
ls models/best.pt

# Or download YOLOv8 pretrained model
yolo task=detect mode=predict model=yolov8n.pt
Video playback is too fast/slow

Solution: The script maintains original FPS. Check your video file's FPS and adjust the wait time in the code if needed.

CUDA/GPU not detected

Solution: Install PyTorch with CUDA support:

pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118

# Verify GPU is available
python -c "import torch; print(torch.cuda.is_available())"
Low detection accuracy

Solutions:

  • Adjust confidence threshold: --conf 0.3
  • Train on more data
  • Use a larger YOLOv8 model (e.g., yolov8m.pt or yolov8l.pt)
  • Ensure good lighting in videos
Permission denied errors

Solution: Run with appropriate permissions or check file/folder access rights:

# On Linux/Mac
chmod +x detect.py

# On Windows, run terminal as Administrator
